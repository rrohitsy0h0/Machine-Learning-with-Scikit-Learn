# Machine Learning with Scikit-Learn

This repository contains data science and machine learning notebooks demonstrating various techniques using Python's scikit-learn library.

## Contents

Jupyter notebook containing two main data analysis tasks:
  1. Data manipulation with Pandas
  2. Breast cancer classification using multiple ML algorithms

## Data Analysis Tasks

### Task 1: Pandas Data Manipulation
- Creating and manipulating dataframes
- Handling missing values
- Data transformation and column operations
- Sorting and filtering data

### Task 2: Breast Cancer Classification
- Data preprocessing
- Train/test split (2:1 ratio)
- Feature scaling using StandardScaler
- Implementation of multiple classification algorithms:
  - Support Vector Machine (SVM)
  - Logistic Regression (LR)
  - Random Forest (RF)
  - Decision Tree (DT)
  - XGBoost
  - Gradient Boosting

## Model Performance

Performance comparison of classification models on breast cancer dataset:
- Evaluation metrics: Accuracy, Precision, Recall, F1 Score
- Visualizations of model performance metrics
- Confusion matrices for each model

## Requirements

- Python 3.x
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- xgboost
